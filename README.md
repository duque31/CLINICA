<h1 align="center">🏥  "Clínica Renova" – Projeto Integrado Multidisciplinar III</h1>
<p align="center">
API desenvolvida em <b>C# com ASP.NET Core</b> para gerenciamento de uma clínica,
permitindo o controle de pacientes, profissionais, atendimentos e tratamentos.
</p>

<hr>

<h2>🚀 Funcionalidades</h2>
<ul>
  <li> Cadastro e autenticação de usuários</li>
  <li> Gerenciamento de profissionais</li>
  <li> Cadastro de pacientes</li>
  <li> Controle de atendimentos</li>
  <li> Gestão de tratamentos</li>
  <li> Validações de dados (CPF, e-mail, etc.)</li>
</ul>

<hr>

<h2>🏗️ Estrutura do Projeto</h2>
<p>O projeto está organizado seguindo boas práticas de arquitetura:</p>

<ul>
  <li><b>Controllers</b> → Responsáveis pelos endpoints da API</li>
  <li><b>Models</b> → Representação das entidades do sistema</li>
  <li><b>DTOs</b> → Transferência de dados entre camadas</li>
  <li><b>Services</b> → Regras de negócio</li>
  <li><b>Data</b> → Contexto do banco de dados</li>
  <li><b>Validations</b> → Validações personalizadas</li>
</ul>

<hr>

<h2>🛠️ Tecnologias Utilizadas</h2>
<ul>
  <li>C#</li>
  <li>ASP.NET Core</li>
  <li>Entity Framework Core</li>
</ul>

<hr>

<h2>🔐 Autenticação</h2>
<p>O sistema utiliza autenticação baseada em login com DTOs:</p>
<ul>
  <li><code>LoginDTO</code></li>
  <li><code>UsuarioDTO</code></li>
</ul>

<hr>

<h2>📌 Endpoints Principais</h2>
<ul>
  <li><code>/api/usuario</code></li>
  <li><code>/api/paciente</code></li>
  <li><code>/api/profissional</code></li>
  <li><code>/api/atendimento</code></li>
  <li><code>/api/tratamento</code></li>
</ul>
